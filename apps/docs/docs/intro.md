# Introduction

Welcome to **streamd** — a streaming-first markdown parser with plugin support, designed for rendering across multiple frameworks.

## What is streamd?

streamd is a generic markdown parser built with a streaming-first architecture. It produces a renderer-agnostic AST that can be consumed by adapters for React, React Native, Vue, and more.

## Key Features

- Streaming-first architecture for real-time markdown rendering
- Plugin system for extensibility
- Renderer-agnostic core parser
- Framework adapters for React, React Native, Vue, etc.
- TypeScript-first with full type safety
