# Changelog

Monorepo-wide notes. Per-package changelogs live under
`packages/<name>/CHANGELOG.md` and are generated by Changesets on
every release.

## Unreleased — 0.2 structural refactor

Cross-cutting refactor across every package. Design:
[ADR 0004](docs/adr/0004-structural-refactor.md).

- Token schema 2: `TOKEN_SCHEMA_VERSION = 2`. `type` fields are
  string literals (`"paragraph"`, `"code_block"`, …). 20 tokens in
  total; `HtmlBlock`, `HtmlInline`, `Softbreak` are absent. Raw HTML
  in source is dropped at block level and kept as literal text
  inline. `CodeBlock` carries `lang`; no separate `info` field.
- Plugins return token objects only. No `meta.html` channel, no
  `allowDangerousMetaHtml`. Plugins annotate typed `meta` fields
  (`highlight`, `frontmatter`, `attrs`, `id`, `className`, `rel`,
  `target`).
- Renderers own rendering. `@streamd/html`, `@streamd/react`,
  `@streamd/react-native` each ship a default component per token
  type plus a `components={}` override map keyed by snake_case
  `TokenTypeValue` discriminants. Default `code_block` emits styled
  spans / `<Text>` from `meta.highlight`; default math renders raw
  TeX in monospace.
- `@streamd/react/streaming` and `@streamd/react-native/streaming`
  provide `<StreamingRevealProvider>`, `<Words>`, `useShouldStream`,
  and sixteen animation presets (fade, fade-up, fade-down,
  slide-in-left, slide-in-right, slide-up, slide-down, scale-up,
  scale-down, blur, blur-fade, blur-up, typewriter, shimmer, ripple,
  none). Native uses the core `Animated` API; web uses CSS
  keyframes.
- Parser adds `shouldEmitSpace(prev, next)` for deterministic block
  spacing and a list-merge streaming fix that combines list items
  arriving in separate chunks into a single `ListToken`.
- `@streamd/plugin-shiki` annotates `CodeBlock.meta.highlight` with
  structured `HighlightData` (`ThemedSegment[][]`).
- `@streamd/plugin-katex` is no longer part of the monorepo — math
  rendering is a component-override concern.
- Streaming throughput +50–77% on the mixed-content benchmarks from
  an incremental list-merge fix.

## Development

- `scripts/docker.sh` runs the full pipeline inside a pinned Debian
  Bookworm image (glibc 2.36, Node 22). Needed on Linux hosts
  shipping glibc < 2.28 where the rolldown / biome native bindings
  cannot load directly. See the "Containerised builds" section in
  the root README.
